# Timestamp Logger

A simple VS Code extension to add timestamps in your files.

## Features

- Add a timestamp at the current cursor position.
- Automatically append a timestamp when you save a file (optional).

## Commands

- `Add Timestamp`: Insert the current timestamp at the cursor.
- `Enable Auto Log`: Enable automatic timestamp logging on file save.
- `Disable Auto Log`: Disable automatic timestamp logging.

## How to Use

1. Install the extension.
2. Use `Ctrl+Shift+P` to open the Command Palette.
3. Run the desired command, such as `Add Timestamp`.
